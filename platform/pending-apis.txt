Locust 压测监控平台 — 待接入接口清单
=====================================
生成时间：2026-05-29
更新说明：2026-05-30 根据仪表盘/测试任务已实现能力同步勾选
Locust 版本参考：2.44.0
API 基址：{locustApiBase}（开发环境默认 /locust-api 代理至 VITE_LOCUST_URL）

一、已接入的 Locust 原生接口
----------------------------
[√] GET  /stats/requests          — KPI 卡片、API 统计表、失败明细、图表（轮询 2s，total_rps 与原生 WebUI 图表一致）
[√] GET  /exceptions              — 异常统计表
[√] GET  /logs                    — 底部日志预览（master 最近 5 条）
[√] POST /swarm                   — 测试任务页「立即执行」（经 /platform/swarm，含 shape_class、user_classes、run_time 等）
[√] GET  /stop                    — 顶部「停止运行」按钮
[√] GET  /stats/requests/csv      — 导出：请求统计 CSV
[√] GET  /stats/failures/csv      — 导出：失败明细 CSV
[√] GET  /exceptions/csv          — 导出：异常 CSV
[√] GET  /stats/report?download=1 — 导出：HTML 报告

二、Locust 原生接口 — 尚未接入
------------------------------
[√] POST /swarm（完整参数）
    - host：目标 Host 覆盖（platform/swarm 支持）
    - shape_class：LoadTestShape 选择（已对接 shapes/）
    - user_classes：UserClass 多选（已传 user_classes）
    - run_time：执行时长（已支持）
    - profile、其他 parsed_options 透传字段：仍未接入

[ ] GET  /stats/reset
    说明：重置统计数据，可在「新任务开始前」或工具栏提供按钮

[√] GET  /stats/requests 图表导出（客户端）
    - 单图 PNG（ECharts getDataURL）
    - 图表时序 CSV（客户端累积 history）
    - 说明：非 Locust 原生路由，由平台前端实现

[ ] GET  /stats/requests_full_history/csv
    说明：需 Locust 启动时启用 stats_history_enabled；用于完整历史曲线导出

[ ] GET  /tasks
    返回：per_class / total 任务执行比例
    说明：可用于「场景/task 占比」可视化

[ ] GET  /worker-count
    说明：分布式模式下展示 Worker 数量（Master 节点）

[ ] GET  /stats/requests 中的 workers 字段
    说明：Master 模式下 Worker 列表（id、state、user_count、cpu、memory）

[ ] POST /user
    说明：运行时动态修改 User 类属性（Locust 高级用法）

[ ] GET  /login、Web 认证
    说明：若 Locust 启用 --web-login，前端需增加登录页与 Cookie 会话

[ ] GET  /metrics（本项目扩展）
    说明：common/metrics.py 暴露的 Prometheus 指标；可与 Grafana Tab 联动

五、前端技术说明
----------------
- 技术栈：React + Ant Design + ECharts + html2canvas
- 图表：轮询 GET /stats/requests（2s），使用 total_rps / total_fail_per_sec 绘制（与 Locust 原生 WebUI 实时图表一致）
- 图表历史：客户端按会话累积采样点（非服务端 stats.history 主路径）
- 开发跨域：vite.config.ts 配置 /locust-api 代理至 VITE_LOCUST_URL
